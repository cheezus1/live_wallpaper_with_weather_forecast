Android live wallpaper with weather forecast

  = This project is developed by Bozhidar Nikolov and Kiril Kostadinov for the subject "Technology of programming", students of "Technological School Electronic Systems", with Kiril Mitov as their teacher.

  = The project represents an android application, which includes:
    - interactive user interface with various functionalities
    - hourly weather forecast for the current day
    - weather forecast for the next number of days, selected from the user
    - live wallpaper, dependent on the weather at the user's current location.
